# Data Classification Matrix

| Data Type | Example | Classification | Access Basis | Storage Rules | Sharing Rules | Monitoring / Evidence |
|---|---|---|---|---|---|---|
| Public | Marketing site content | Public | Open | Standard approved systems | Public distribution allowed | Change history |
| Internal | Internal process docs | Internal | Employee need-to-know | Corporate systems | Internal only | Access logs |
| Proprietary Technical Data | Design docs, architecture diagrams | Restricted Internal | Role-based | Approved engineering systems | Approval for external sharing | Repo access logs, DLP events |
| Export-Controlled Technical Data | RF designs, sensitive schematics | Export-Controlled | U.S. person / approved personnel only | Segmented environment only | No unapproved external sharing | Access approvals, audit logs |
| CUI | Customer or government-controlled data | CUI | Contractual and role-based need-to-know | U.S.-restricted approved systems | Controlled transmission only | Access logs, encryption evidence |
