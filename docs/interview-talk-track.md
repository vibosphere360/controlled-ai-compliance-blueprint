I wanted to demonstrate how I think about compliance as an operating system, not just a policy set.

## For Field AI
This project shows how I would approach controlled technical information, source code handling, access segmentation, and technology control planning in a fast-moving AI company.

## For AnySignal
This project shows how I would approach defense-adjacent compliance leadership across ITAR/EAR, NIST-style controls, environment segregation, and scalable operational governance.
